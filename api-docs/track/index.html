    tree 

    tree(
    treedata: str | Path | StringIO | Tree,
    *,
    format: str = "newick",
    outer: bool = True,
    use_branch_length: bool = False,
    leaf_label_size: float = 0,
    innode_label_size: float = 0,
    leaf_label_margin: float = 0.5,
    label_formatter: Callable[[Clade], str] | None = None,
    node_color_list: list[tuple[list[str], str]] = [],
    line_kws: dict[str, Any] = {},
    text_kws: dict[str, Any] = {}
) -> None

    Plot tree

    It is recommended that the track(sector) size be the same as the number of leaf nodes in the tree, to make it easier to combine with bar and heatmap.

    PARAMETER DESCRIPTION
    treedata

    Phylogenetic tree data (File-like object or Tree object)

    TYPE: str | Path | StringIO | Tree

    format

    Tree format (e.g. newick, nexus, phyloxml, ...)

    TYPE: str, optional DEFAULT: 'newick'

    outer

    If True, plot tree on outer side. If False, plot tree on inner side.

    TYPE: bool, optional DEFAULT: True

    use_branch_length

    If True, tree branch length is used for plot tree. If False, plot tree with ultrametric tree style.

    TYPE: bool, optional DEFAULT: False

    leaf_label_size

    Leaf node label size. By default, size=0 (No display).

    TYPE: float, optional DEFAULT: 0

    innode_label_size

    Internal node label size. By default, size=0 (No display).

    TYPE: float, optional DEFAULT: 0

    leaf_label_margin

    leaf node label margin size (Radius unit)

    TYPE: float, optional DEFAULT: 0.5

    label_formatter

    User-defined label format function. (e.g. lambda node: node.name.replace("_", " "))

    TYPE: Callable[[Clade], str] | None, optional DEFAULT: None

    node_color_list

    Tree node & color setting list. If multi nodes are set, MRCA(Most Recent Common Ancestor) node of target nodes is set. (e.g. [(["node1"], "red"), (["taxa1", "taxa2"], "blue"), ...])

    TYPE: list[tuple[list[str], str]] DEFAULT: []

    line_kws

    Patch properties (e.g. dict(ec="red", lw=1, ls="dashed", ...)) https://matplotlib.org/stable/api/_as_gen/matplotlib.patches.Patch.html

    TYPE: dict[str, Any], optional DEFAULT: {}

    text_kws

    Text properties (e.g. dict(color="red", ...)) https://matplotlib.org/stable/api/_as_gen/matplotlib.axes.Axes.text.html

    TYPE: dict[str, Any], optional DEFAULT: {}
